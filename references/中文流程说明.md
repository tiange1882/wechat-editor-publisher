# 微信公众号本地免费发布流程说明

这份说明用于复用今天搭好的流程：先润色文章，再排版配图，最后通过微信官方接口把文章存入微信公众号草稿箱。

这个流程不依赖 `md2wechat_key`，也不需要购买 md2wechat 的付费 API。它直接调用微信官方接口。

## 触发方式

以后可以直接用这些说法触发这个流程：

```text
用微信公众号流程处理这篇文章，帮我润色、排版、配图，并存到草稿箱。
```

```text
处理这篇公众号文章，存草稿。
```

```text
把下面这篇文章做成公众号草稿。
```

```text
帮我做公众号：润色、配图、排版、上传草稿箱。
```

也可以只触发其中一部分：

```text
只润色，不发布。
```

```text
只做小红书图文卡片。
```

```text
只生成公众号封面和文中配图。
```

```text
不要上传，先给我预览。
```

默认完整流程是：润色文章 -> 整理标题和摘要 -> 配封面图和文中图 -> 生成 Markdown -> 本地预览 -> 创建微信公众号草稿。正式上传草稿箱前，应该先检查预览结果。

## 一、适用场景

适合这些需求：

- 润色微信公众号文章
- 生成或整理公众号封面图
- 插入文中配图
- 把 Markdown 文章转成微信可用 HTML
- 上传封面和文中图片
- 创建微信公众号草稿

不适合这些需求：

- 直接群发文章
- 绕过微信公众号后台审核
- 不具备公众号官方接口权限的账号

## 二、文章文件要求

建议文章文件命名为：

```text
article-wechat.md
```

文件开头使用 frontmatter：

```yaml
---
title: "文章标题"
author: ""
digest: "这里写 128 字以内摘要"
---
```

正文里可以使用本地图片：

```markdown
![封面](output/wechat/wechat-cover.png)

![文中配图](output/wechat/01-example.png)
```

注意：

- `title` 已经在 frontmatter 里时，正文里不要再放一个完全相同的一级标题。
- 图片路径必须真实存在。
- 文件必须保存为 UTF-8。

## 三、配置 AppID 和 AppSecret

脚本默认读取：

```text
C:\Users\你的用户名\.config\md2wechat\config.yaml
```

配置格式：

```yaml
wechat:
  appid: wx开头的AppID
  secret: 你的微信公众号AppSecret
```

打开配置文件：

```powershell
notepad "$env:USERPROFILE\.config\md2wechat\config.yaml"
```

不要把 AppSecret 发到聊天窗口里。只在本机配置文件里填写。

## 四、API IP 白名单

如果脚本报：

```text
40164
```

说明当前电脑访问微信接口时使用的公网 IP 不在白名单里。

处理步骤：

1. 看脚本错误里显示的微信识别 IP。
2. 打开微信公众号后台。
3. 进入：

```text
设置与开发 -> 基本配置 -> API IP白名单
```

4. 把脚本提示的 IP 加进去。
5. 保存后等 1-5 分钟，再重新运行脚本。

## 五、先生成本地预览

先不要直接上传，先跑 dry-run：

```powershell
Set-Location "D:\微信公众号编辑"
node .\publish-wechat-official.mjs article-wechat.md --cover output/wechat/wechat-cover.png --dry-run
```

成功后会生成：

```text
wechat-official-preview.html
```

打开这个 HTML 检查：

- 标题是否正确
- 段落是否舒服
- 图片是否都在
- 有没有乱码
- 是否有重复标题

## 六、正式创建微信公众号草稿

确认预览没问题后运行：

```powershell
Set-Location "D:\微信公众号编辑"
node .\publish-wechat-official.mjs article-wechat.md --cover output/wechat/wechat-cover.png
```

脚本会自动做：

1. 获取 `access_token`
2. 上传封面图为永久 thumb 素材
3. 上传正文图片
4. 把 Markdown 转成微信兼容 HTML
5. 调用 `draft/add` 创建草稿

成功后，到微信公众号后台草稿箱检查。

## 七、常见错误

### 40164

当前公网 IP 不在 API IP 白名单。

把脚本提示的 IP 加到公众号后台白名单。

### 40001

AppID 或 AppSecret 不正确。

重新检查配置文件，必要时去公众号后台重置 AppSecret。

### 48001

接口未授权。

常见原因：

- 公众号未认证
- 当前账号类型没有素材/草稿接口权限
- 接口能力未开放

### 封面上传失败

微信封面 thumb 素材可能有大小限制。先把封面压缩到更小，通常建议低于 64KB。
